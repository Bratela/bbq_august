#Simple app BBQ.
##Created with Goodprogrammer.ru

How to run app:
 * copy last version : git clone https://github.com/Bratela/bbq_august.git
 * go to bbq folder
 * run command : bundle exec
 * run command : rake db:migrate
 * and start server : rails s
